# BUTA TCG v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add multi-tournament support, per-tournament payment alias with copy button, an optional ≤100-char comment on registration, and authorship marks, per the approved spec at `docs/superpowers/specs/2026-06-10-buta-tcg-v2-design.md`.

**Architecture:** Extends the v1 static site (HTML + Tailwind CDN + vanilla ES modules + Google Sheets/Apps Script backend) in place. `js/data.js` gains a plural selector (`pickProximos`); `torneos.html`/`js/inscripcion.js` are reworked to render N tournament cards plus a single form with a tournament selector; the Apps Script gains a `counts` action and a `comentario` column. All changes are backward-compatible with the live sheet until BUTA migrates it.

**Tech Stack:** unchanged from v1 (Node 24 for tests/tooling; `npm test` = `node --test tests/*.test.mjs`; serve on :3000; screenshots via `node screenshot.mjs <url> [label] [WIDTHxHEIGHT]`).

**Site is LIVE:** production is https://l4gash.github.io/buta-tcg-web/ reading the real (restricted) sheet via published CSVs and a deployed Apps Script. Tasks must not break the live contract until the deploy task. Work on a feature branch off `main`; do NOT push until the final deploy task.

**Authorship-marks confidentiality:** this plan is public in the repo. Task 6 deliberately does NOT document the marks; the controller provides those instructions out-of-band (per spec §4). This is an intentional, approved exception to the no-placeholders rule.

const fmtFecha = (iso) => {

      <div class="mt-4 flex flex-wrap items-center gap-2 rounded-lg border border-borde bg-noche/60 px-3 py-2 font-body text-sm text-humo">

function renderTorneos() {
  $('lista-torneos').innerHTML = proximos.map(tarjetaTorneo).join('');
}

function renderSelector() {
  const sel = $('torneo-select');
  const previo = sel.value;
  sel.innerHTML = proximos.map((t) => {
    const lleno = estaLleno(t);
    return `<option value="${esc(t.nombre)}" ${lleno ? 'disabled' : ''}>${esc(t.nombre)}${lleno ? ' — Cupo lleno' : ''}</option>`;
  }).join('');
  const previoValido = proximos.some((t) => t.nombre === previo && !estaLleno(t));
  if (previoValido) sel.value = previo;
  else {
    const libre = proximos.find((t) => !estaLleno(t));
    if (libre) sel.value = libre.nombre;
  }
}

async function consultarConteos() {
  if (!APPS_SCRIPT_URL) return null;
  try {
    const res = await fetch(`${APPS_SCRIPT_URL}?action=counts`);
    const data = await res.json();
    return data.ok && data.counts ? data.counts : null;
  } catch { return null; }
}

// Copiar alias (delegado: las tarjetas se re-renderizan)
